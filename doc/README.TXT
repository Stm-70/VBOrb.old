Welcome to VBOrb
----------------

- VBOrb is an Object Request Broker.
- VBOrb is written entirely in Visual Basic 5.
- VBOrb comes with an CORBA IDL to VB compiler called IDL2VB.
  The compiler is written in Java.

To read more about VBOrb please open the file index.html with a browser
of your choice.
